Etiquetas contenedor

Las etiquetas div y span en Html son elementos no semanticos,lo que significa que no aportan ningun significado especifico sobre su contenido.

Se utilizan pricipalmente como contenedores generales para agrupar elementos, aplicar estilos con CSS o funcionalidades con JS.
